# CCAO-F Trainer Course

Training course for the **Claude Certified Associate – Foundations (CCAO-F)** exam: 7 domain modules with trainer notes and practice questions, an exam overview with study/delivery plans, and a 60-question blueprint-weighted mock exam.

**Read it as a website (recommended):** enable GitHub Pages on this repo, then browse the rendered site — or view [`index.md`](index.md) for the table of contents.

All exam facts come from the official CCAO-F Exam Guide v1.0 (Anthropic Partner Academy); product content cites official Anthropic documentation. All practice questions are original items — no NDA-protected exam content. Independent, unofficial study resource; not affiliated with or endorsed by Anthropic.
